# Bottorff et al. 2026 Manuscript Code

This repository houses code used to analyze and generate figures from all data presented in Bottorff et al., 2026.

## Analysis scripts

Scripts detail analyses required to make figures. Large raw data and saved data objects referred to in each script are available for download via [Box](LINK).
